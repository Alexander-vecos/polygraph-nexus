# PolyGraph Nexus - Frontend

Это фронтенд-часть производственной системы управления для полиграфии.

## Технологии
- React 18 + TypeScript
- Vite 7 + PWA (vite-plugin-pwa)
- Supabase для аутентификации и БД
- Tailwind CSS (если используем)
- Zustand для состояния

## Запуск
npm install
npm run dev

## Сборка
npm run build
npm run preview
